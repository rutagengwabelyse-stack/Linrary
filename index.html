Real Problem • Real Solution

In Kigali, many young people in our sectors have limited access to books due to cost and few community libraries. Kigali BookHub solves this by providing a free, digital-first book selection platform with community-driven suggestions.

  • ✅ Realistic simulated data from 3 Kigali districts
  • ✅ Bootstrap-powered responsive design
  • ✅ Form validation prevents invalid entries
  • ✅ Facebook + X API integration for live feeds
  • ✅ Scalable: Just add new districts/books in JS data array
